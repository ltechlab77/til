## カプセル化
フィールドやメソッドにアクセス制御をかける  

アクセス制御レベル
```
//厳しい順
private
  └ このクラスの中だけ

package-private（この指定方法は何も書かないこと）
  └ 同じパッケージだけOK

protected
  └ 同じパッケージ
  └ 自分を継承した子クラス

public
  └ どこからでもOK
  ```

### getter
privateなフィールドの値を、外部から安全に取得するためのメソッド　　
```
public フィールドの型 getフィールド名() {
    return this.フィールド名;
}
```
private設置してあるフィールドはほかのクラスからそのフィールドは見えない、使えないからgetterを介してフィールドを取り出す  

### setter
外部からフィールドの値を変更するためのメソッド。
```
public void setフィールド名 (フィールドの型　任意の変数名) {
    this.フィールド名 = 任意の変数名;
}
```
