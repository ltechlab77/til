## 静的フィールド

クラスにひもづく変数のこと。

インスタンス（new したオブジェクト）ごとに持つんじゃない  
クラス全体で1つだけ共有される

```java
class Hero {
    static int gold = 100;//静的フィールド
}
```

インスタンスフィールド → 各勇者の持ち物  
静的フィールド → パーティ全体の所持金   
勇者Aが増やしても、勇者Bが見ても同じ数字。 

<br>

```java
Hero.gold += 50;
System.out.println(Hero.gold);
```
アクセス方法  
`クラス名.静的フィールド名`

よくある使い道
- 全体で共通の値
- 税率
- 設定値
- カウンター
- 定数（static final）
多くの場合finalやpublicを伴って宣言され定数を機共有するために利用される

## 静的メソッド
```java
class Hero {
    static void greet() {
        System.out.println("勇者たちよ、集まれ！");//静的メソッド
    }
}
```
呼び出し方
`Hero.greet();`
クラス名.メソッド名();

- メソッド自体がクラスに属するようになる
- インスタンスにメソッドの分身が準備される
- インスタンスを生み出さなくても呼び出せる
