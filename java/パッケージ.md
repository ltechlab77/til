# パッケージ  

クラスやインターフェースを整理するためのフォルダ  
`package 所属させたいパッケージ名;`
ソースコードの先頭に書く  

目的  
- クラス名の衝突某防止  
- コード整理  
- アクセス制御  

### FQCN  
`パッケージ名.クラス名`  
`java.util.ArrayList`  
パッケージ名：java.util  
クラス名：ArrayList  
importを使わずに直接指定できる  
`java.util.ArrayList list = new java.util.ArrayList();`


### import
`import パッケージ名.クラス名`  
ソースコードの先頭、packageの後

### API  
javaが標準で提供しているクラスやメソッドの集合  
代表的なjava標準API
- java.lang : 自動でインポートされる基本クラス群 (String,Systesm,Math,)
- java util : 便利なユーティリティ(List,ArrayList,Map,Date)
- java.io/java.nio : 入出力処理(file,Path)
- java time : 新しい日付API(LocalDate,ZonedDateTime)
- java.net : ネットワーク(URL,Socket)
